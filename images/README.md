# 🖼️ Lily Pi — Images

This folder contains visual assets referenced in the README and documentation.

| File | Description |
|------|-------------|
| `hardware-assembly.png` | Hardware layout diagram / assembly photo placeholder |
| `ui-mockup.png`         | OLED HUD display layout mockup placeholder |

## Contributing Images

- Preferred format: **PNG** (lossless, good for diagrams)
- Resolution: **800 × 600** or larger for assembly photos; **400 × 300** for icons
- Name files using `kebab-case.png`
- Update this table when adding new images
