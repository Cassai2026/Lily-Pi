## 🚀 Project Structure
* **`/src`**: The "Eldest's Domain." Contains `lilieth_kernel.c` and Assembly bootloader files.
* **`/scripts`**: The "19yo's Domain." LUA logic and the Mental Printer protocols.
* **`/ui`**: The "13yo's Domain." HTML/CSS/JS for the Iris interface.
* **`/docs`**: The Nanna Protocol and detailed SDG Mandate whitepapers.

## 🛠️ Getting Started (The Alpha-Omega Launch)
1. **The Bootloader**: Our custom Assembly code wakes the hardware and points the CPU to the kernel.
2. **The Kernel**: Written in C, handling memory management and the Sovereign Node distribution.
3. **The Broadcast**: Upon `kmain()` initialization, the kernel broadcasts the NGO Mandate to the console.
