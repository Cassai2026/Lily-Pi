# Lily-Pi AR HUD
## Modular Augmented Reality System for Raspberry Pi 5

### Project Structure
- **core/**: Hardware drivers and AI bridge logic.
- **main.py**: System entry point.
- **config.yaml**: User-defined thresholds and paths.
- **logs/**: Local telemetry audit trail.

### Deployment
1. pip install -r requirements.txt
2. python main.py
